# 💬 Real-Time Chat Application

A real-time chat app built using Node.js, Express, and Socket.IO.

## 🚀 Features
- Real-time messaging
- Multiple users
- Username support
- Instant updates

## 🛠️ Tech Stack
- Node.js
- Express
- Socket.IO
- HTML, CSS, JavaScript

## ▶️ Run Locally
```bash
npm install
npm start
